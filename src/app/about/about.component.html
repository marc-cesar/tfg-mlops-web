About this project

This web page is part of a project developed by Marc Cesar, a student at the © UPC Universitat Politècnica de Catalunya BarcelonaTech for the Bachelor Degree in Informatics Engineering. The aim is to contribute to the academic field by applying theoretical knowledge in a practical environment.

The core of this project is a prototype system that uses MLOps (Machine Learning Operations) technology to optimize the machine learning lifecycle, from data preparation to model training, deployment, and monitoring. Learn more about MLOps at ml-ops.org.

More specifically, this is the prototype of a Credit Prediction System that uses machine learning models to predict the credit risk of a customer. The system is designed to be used by a bank to automate the credit risk assessment process and improve the efficiency of the loan approval process.
